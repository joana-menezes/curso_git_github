# GDPR / LGPD
1. GDPR: any data
2. LGPD: dados pessoais (infromações relacionadas a pessoa)

*Sensitive data* termo nao usado > porém podem ser incluidos dados financeiros e dados onde podem projudicar ou comprometer outrem.
